>>使用说明：

                $('选择器').pagination({
                          total   : 100,       //必填，数据总共有多少条
                          doPaging :  function(page,perPage){}, // 必填，点击分页后的回调.改回调有2个参数，分别是当前页码和每页数量
                          perPage: 10 ,   //可选， 每页显示多少条记录, 默认10
                          showPages : 7 , //可选，总共要显示多少个页码。 默认7
                          isShowFirstPage : false, //可选，是否显示首页按钮 默认 false
                          isShowLastPage ： false, //可选，是否显示最后页按钮 默认 false  
                          isShowTotalNum : false, //可选，是否显示全部数量 默认 false
                          isShowPageItems : true, //可选，是否显示分页按钮 默认 true
                          isShowPrev : true,     //可选，是否显示上一页按钮 默认 true
                          isShowNext :true ,     //可选，是否显示下一页按钮 默认 true
                          isShowSelect : true    //可选，是否显示页码选择器 默认 true
                     })

基础功能 

 JS代码：
            $('#demo').pagination({
                total:100,
                doPaging:function(page,num){
                    $('.test').eq(0).html(`当前页是${page}每页数量${num}`);
                }
            })

每页数量，显示页面数量 

        JS代码:
        $('#demo1').pagination({
            total:100,
            perPage:3,
            showPages: 9,
            doPaging:function(page,num){
                $('.test').eq(1).html(`当前页是${page}每页数量${num}`);
            },
          
        })
配置 perPage(每页数量) : 3 默认值:10 , showPages(显示多少个页) : 9 默认值:7

显示 首页、最后一页 

        JS代码：
        $('#demo2').pagination({
            total:100,
            isShowFirstPage : true,
            isShowLastPage : true,
            doPaging:function(page,num){
                $('.test').eq(2).html(`当前页是${page}每页数量${num}`);
            },
           
        })
配置 isShowFirstPage : true 默认值:false , isShowLastPage: true 默认值:false

隐藏 页码下拉框 

        JS代码：
        $('#demo3').pagination({
            total:100,
            isShowSelect  : false,
            doPaging:function(page,num){
                $('.test').eq(3).html(`当前页是${page}每页数量${num}`);
            },
           
        })
配置 isShowSelect : false 默认值: true

隐藏 上一页，下一页 

        JS代码：

配置 isShowPrev: false 默认值: true , isShowNext : false 默认值:true
